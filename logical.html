2 Logical

Use logical values for characteristics that are either TRUE or FALSE. Note that if logical elements can also have an NA value if the observation is missing.

# evaluate as logical, test whether 1 is greater than two
a <- 1 > 2

# logical NA
age_john <- 39
age_jane <- NA

2.1.1 Vector

A vector is an ordered set of elements of one or more elements of the same data type. Order is important, i.e.,

1, 2, 3 is not the same as 1, 3, 2

R will maintain the order of elements in vectors unless a process is initiated that changes the order of those elements
